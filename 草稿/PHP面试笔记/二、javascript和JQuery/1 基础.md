### javascript 基础语法
#### 变量的定义
- 变量必须以字母开头，也能用_ 和 $ 开头
- 使用var 来声明变量 （现在最好用let）
- 可以在一条语句中声明多个变量
- 未使用值来声明的变量，默认值是undefined
- 重新声明变量，该变量值不会丢失
```
var a = 1;
var a;
```
